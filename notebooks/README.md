# Notebook 入口

当前项目主流程使用下面 4 个 notebook：

| Notebook | 用途 |
|---|---|
| `01_data_collection_and_observing.ipynb` | 目标元数据、观测准备和已有产物盘点。 |
| `02_spectral_analysis_pipeline.ipynb` | 主光谱分析 pipeline，生成可复现表格和报告用图。 |
| `03_tardis_modeling_optional.ipynb` | 可选 TARDIS 定性比较，只作谱线识别和谱形解释辅助。 |
| `04_project_report.ipynb` | P2Rp2 风格报告 notebook，汇总图表、解释和结论。 |

旧版探索 notebook 保留在 `legacy/`，用于追溯早期尝试，不再作为正式流程入口。
